# UnivsAI Gate Service Web — QA 테스트 케이스

> 버전: v0.0.2
> 작성일: 2026-03-05
> 대상: QA 테스터

---

## 테스트 환경

| 항목 | 내용 |
|------|------|
| 브라우저 | Chrome 최신, Safari 최신 |
| 해상도 | 1280px 이상 (데스크탑) |
| 계정 | QA 전용 계정 사전 준비 필요 |
| 테스트 서버 | 개발 서버 (`npm run dev`) 또는 스테이징 서버 |

---

## 테스트 진행 방법

- **Pass** : 기대 결과와 일치
- **Fail** : 기대 결과와 다름 → 이슈 등록
- **N/A** : 해당 없음 (환경 미구성 등)
