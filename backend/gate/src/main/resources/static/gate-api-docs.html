API Reference

Gate Service API

얼굴 인식 및 팜 정맥 인식 기반 e-KYC 게이트 서비스의 공식 API 레퍼런스입니다. 얼굴·팜 특징점 관리, 1:1 확인, 1:N 매칭, 라이브니스 감지 기능을 REST API로 제공합니다.

Base URLhttps://api.univsgate.com
Versionv1.0
ProtocolHTTPS · REST
FormatJSON / Multipart

소개

UnivsGate API는 프로젝트별 API Key를 기반으로 얼굴 인식 및 팜 정맥 인식 기반 e-KYC 기능을 외부 서비스에 통합할 수 있는 RESTful API입니다. JWT 인증과 API Key 인증을 함께 사용하며, 파일 업로드가 필요한 엔드포인트는 multipart/form-data 형식을 사용합니다.

API를 사용하기 전에 관리 페이지에서 회원가입 및 로그인 후 프로젝트를 생성하고 API Key를 발급받으세요. API Key는 모든 e-KYC 요청에 필수 인증 수단입니다.

모든 API 응답은 {"success": boolean, "data": ..., "errors": ...} 형식의 통일된 구조로 반환됩니다.

지원 생체인증 모듈

👤얼굴 인식 (Face)

얼굴 특징점 등록/관리 및 매칭(1:1 확인, 1:N 매칭, 라이브니스)을 지원합니다.

Base: /api/v1/feature/face

✋팜 정맥 인식 (Palm)

팜 정맥 특징점 등록/관리 및 매칭(1:N 매칭, 라이브니스)을 지원합니다.

Base: /api/v1/feature/palm

Base URL

Base URLhttps://api.univsgate.com/api/v1

인증 방식

특징점 관리 및 매칭 API는 두 가지 인증 헤더가 모두 필요합니다. 로그인 API는 인증이 불필요합니다.

🔐JWT Bearer Token

로그인 후 발급된 accessToken을 Authorization 헤더에 포함합니다.

Authorization: Bearer {accessToken}

🔑API Key

프로젝트별로 발급된 API Key를 X-Api-Key 헤더에 포함합니다.

X-Api-Key: {apiKey}

응답 구조

성공/실패 모두 HTTP 200으로 응답합니다. success 필드로 처리 결과를 구분합니다.

✅ 성공 응답

{
  "success": true,
  "data": { ... },
  "errors": null
}

❌ 오류 응답

{
  "success": false,
  "data": null,
  "errors": {
    "code": "ERROR_CODE",
    "message": "..."
  }
}

타임존 처리

응답의 날짜/시간 필드는 기본적으로 UTC 기준입니다. Accept-TimeZone 헤더로 원하는 타임존을 지정하면 자동 변환됩니다.

ExampleAccept-TimeZone: Asia/Seoul

다국어 처리

한국어/영어를 지원합니다. Accept-Language 헤더로 원하는 언어를 지정하면 자동 변환됩니다.

ExampleAccept-Language: ko
