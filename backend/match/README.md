# Match Server

![Version](https://img.shields.io/badge/version-1.0.0-blue)

## 개요

---

**Match server**는 사용자 특징점을 관리하고 1:1 또는 1:N 매칭을 지원하는 서버입니다.  

## 사용 범위

---

- 얼굴 서비스

## 사용 방법

---

매처 서버는 별도의 사용 방법을 지원하지 않습니다.  
모든 얼굴 요청은 중개 서버인 얼굴 서비스를 통해서 제공하고 있습니다.
