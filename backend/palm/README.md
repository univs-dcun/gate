# Palm Service

![Version](https://img.shields.io/badge/version-1.0.0-blue)

## 개요

---

**Palm service**는 사용자 손바닥 이미지를 매처 서버, 추출 서버에 라우팅하는 역할을 담당합니다.  
추가적으로 위 서버들에 요청을 처리하고 반환되는 결과를 데이터베이스에 로깅합니다.

## 사용 방법

---

- API URL (Swagger): https://develop.univs.ai:18090/swagger-ui.html
- 접속 후 상단 우측에 Select a definition 박스에서 필요한 서비스 선택
